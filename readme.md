# clipboard-cli

> Access the system clipboard (copy/paste) - cross-platform

Supports: macOS, Windows, Linux, OpenBSD, FreeBSD, Android with [Termux](https://termux.com).

## Install

```sh
npm install --global clipboard-cli
```

## Usage

```
clipboard --help
```
